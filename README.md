# beneplus.in

Marketing site for **Beneplus** — an AI-native software studio building mobile, web, blockchain and emerging-tech products.

Live at <https://beneplus.in>.

## Stack

Static HTML + React (via CDN) + Babel-in-browser. No build step. Open any `.html` directly, or serve the folder with any static server.

## Pages

- `index.html` — home
- `about.html` — about the studio
- `services.html` — eight practices
- `industries.html` — sixteen industries
- `portfolio.html` — selected work
- `contact.html` — schedule a call
- `brand.jsx`, `site.jsx` — shared React components (logo, nav, footer, hooks)
